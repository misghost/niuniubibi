# CRM 续期提醒管理程序

这是一个本地可运行的 Web 版 CRM 提醒工具，直接读取 `/Volumes/D/wps同步/AI项目信息/浙江.xlsx`，按以下规则生成提醒：

- 读取 `S` 列“报告落款日”
- 将 `报告落款日 + 1 年` 视为续期日
- 将 `续期日前 3 个月` 视为开始提醒日
- 被提醒人固定使用 `签约人(AC)`
- 同一客户如果有多年度报告，只保留 `最新报告落款日` 这一条来计算提醒

## 运行方式

```bash
cd /Volumes/D/编程/crm-reminder
python3 app.py
```

浏览器打开：

```text
http://127.0.0.1:8765
```

## 功能说明

- 客户提醒列表：展示已逾期、应提醒、待进入提醒期客户
- 点击客户名称：右侧查看该客户完整 Excel 详情
- 搜索与筛选：可按客户名、项目名、项目编号、被提醒人、状态筛选
- 重点提醒人：按当前筛选结果中的紧急客户数量聚合
- 重新加载 Excel：源文件有更新时，无需重启程序
- 桌面提醒：在 `localhost` 打开后，可启用浏览器桌面通知

## 自定义 Excel 路径

如果以后换文件，可以这样启动：

```bash
CRM_EXCEL_PATH="/你的/Excel路径.xlsx" python3 app.py
```

## 说明

- 当前版本不依赖第三方 Python 包
- 目前是“系统内提醒 + 桌面通知”模式
- 如果你后面想接入企业微信、钉钉或邮件推送，可以在现有 API 基础上继续扩展
